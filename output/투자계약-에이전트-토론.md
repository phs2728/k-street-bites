## 1.1 💰 CFO-TAX 에이전트 (재무/세무)

### 🎯 핵심 발언

> "두 옵션 모두 투자자에게 합리적 수익을 보장하면서도, 대표의 $3,000 급여를 반드시 보호해야 합니다. 사업이 곧 대표의 생계이기 때문입니다."

### 📊 기업 밸류에이션 산정

| 밸류에이션 방법 | 산출 근거 | 기업 가치 |
|--------------|---------|:--------:|
| **DCF (할인현금흐름)** | 3년 FCF 할인율 15% (해외 소규모 외식업 리스크 반영) | **$65,000~75,000** |
| **매출 배수법** | Year 2 예상 매출 $204,000 × 0.3배 (소규모 외식업) | **$61,200** |
| **순이익 배수법** | Year 2 순이익 $38,184 × 2.0배 (고성장 초기 기업) | **$76,368** |
| **종합 Pre-money 밸류에이션** | 3개 방법 평균 | **약 $67,000** |

→ Post-money: $67,000 + $20,000 = **$87,000**  
→ $20,000 투자 시 적정 지분: **23.0%** (= 20,000 / 87,000)

### 📈 Option A: 대출 — 적정 이자율 분석

| 기준금리/비교 대상 | 이자율 | 비고 |
|----------------|:-----:|------|
| 조지아 중앙은행 기준금리 (2025) | 8.0% | NBG 정책금리 |
| 조지아 상업은행 중소기업 대출 | 12~18% | 외국인/무담보 추가 프리미엄 |
| 미국 SBA 소기업 대출 | 7~10% | 비교 참고 |
| **BAM 동역자 간 대출 적정 이자** | **5~8%** | 시장보다 낮지만 실질 보상 |
| 인플레이션 보상 (조지아) | 4~5% | 최소한 이만큼은 보전 필요 |

**CFO-TAX 권고**: **연 6% 고정금리**
- 조지아 인플레이션(4~5%) 상회 → 투자자 실질 수익 보전
- 시장 대출금리(12~18%)보다 상당히 낮음 → 대표에게 유리
- BAM 동역자 관계를 고려한 "은혜의 금리"이지만 무이자는 아님
- 무이자는 투자자에게 불공정하며, 향후 관계 부담 발생 가능

### 📊 대출 이자 비용 시뮬레이션 (연 6%, $20,000)

| 상환 방식 | 총 이자 비용 | 월 상환금 (상환기간) | 총 상환액 |
|---------|:----------:|:-----------------:|:-------:|
| **거치 6개월 + 18개월 균등 원리금** | **$1,236** | 거치: $100 (이자만) / 상환: $1,179 | **$21,236** |
| **거치 6개월 + 24개월 균등 원리금** | **$1,636** | 거치: $100 / 상환: $909 | **$21,636** |
| **거치 12개월 + 18개월 균등 원리금** | **$1,836** | 거치: $100 / 상환: $1,179 | **$21,836** |

**CFO-TAX 추천**: 거치 6개월 + 24개월 균등 상환 → 총 이자 $1,636, 월 상환 $909 → 현금흐름 부담 최소화

### 📊 Option B: 지분투자 — 배당 정책

| 항목 | 제안 |
|------|------|
| **배당 개시 조건** | 누적 순이익이 흑자 전환된 후 (Year 1 적자 보전 후) |
| **배당률** | 순이익의 **20~30%** (단계적) |
| **배당 시기** | **반기 1회** (6월, 12월) — 분기별은 소규모 사업에 행정 부담 |
| **배당 전 우선 사항** | ① 대표 급여 지급 → ② 이연 급여 보전 → ③ 운영자금 확보 → ④ BAM 기금 → ⑤ 배당 |
| **내부유보율** | 최소 70% (성장 재투자용) |

**투자자 수익 시뮬레이션 (지분 23%, 배당 25%)**:

| 연도 | 순이익 | 배당 가능액 | 배당 총액 (25%) | 투자자 몫 (23%) | 누적 수령 |
|------|:------:|:---------:|:-------------:|:--------------:|:--------:|
| Year 1 | ($4,192) | $0 | $0 | **$0** | $0 |
| Year 2 | $38,184 | $33,992* | $8,498 | **$1,955** | $1,955 |
| Year 3 | $77,237 | $77,237 | $19,309 | **$4,441** | $6,396 |
| Year 4 (추정) | $95,000 | $95,000 | $23,750 | **$5,463** | $11,859 |
| Year 5 (추정) | $110,000 | $110,000 | $27,500 | **$6,325** | $18,184 |

*Year 2 배당 가능액 = $38,184 - $4,192(Year 1 적자 보전) = $33,992

**5년 누적 배당: ~$18,184** + **지분 가치 상승** → 투자자 총 ROI 충분

## 1.2 ⚖️ LEGAL-COMPLIANCE 에이전트 (법무)

### 🎯 핵심 발언

> "조지아에서 IE(개인사업자)는 '지분' 개념이 없습니다. 따라서 Option A(대출)는 IE로, Option B(지분투자)는 반드시 LLC로 설립해야 합니다. 이것이 두 옵션의 근본적 차이입니다."

### 📋 법인 구조 비교

| 항목 | IE (개인사업자) | LLC (유한책임회사) |
|------|:-------------:|:-----------------:|
| 세율 | **1%** (매출 500K GEL 이하) | **15%** (배당 시만) = 에스토니아 모델 |
| 지분 구조 | ❌ 불가능 | ✅ 가능 |
| 유한 책임 | ❌ 무한책임 | ✅ 유한책임 |
| 설립 비용 | 20 GEL | 100~200 GEL |
| 설립 기간 | 1일 | 1~2일 |
| 투자자 보호 | 계약으로만 | 법인 구조 + 계약 |
| 적합 옵션 | **Option A (대출)** | **Option B (지분투자)** |

### 📋 Option A: 대출 계약 — 법적 구조

**법적 형태**: IE + 금전소비대차 계약 (Loan Agreement)

**필수 계약 조항**:

| 조항 | 내용 | 법적 근거 |
|------|------|----------|
| 1. 당사자 | 대여인(투자자), 차용인(대표, IE 사업자) | 조지아 민법 |
| 2. 대출 원금 | $20,000 USD | 금액 명시 |
| 3. 이자율 | 연 6% 고정 | 계약 자유 원칙 |
| 4. 상환 스케줄 | 거치 6개월 + 24개월 균등 원리금 | 상환표 별첨 |
| 5. 담보 | **무담보** (BAM 관계 기반 신뢰 대출) | — |
| 6. 조기상환 | 수수료 없이 언제든 조기상환 가능 | 차용인 유리 |
| 7. 연체 조항 | 30일 이상 연체 시 연 3% 추가 연체이자 | 과도하지 않은 수준 |
| 8. 상환 유예 | 월 매출 BEP 70% 미달 시 최대 3개월 유예 가능 | **대표 보호** |
| 9. 불가항력 | 전쟁, 팬데믹, 자연재해 시 상환 중단 | 양측 보호 |
| 10. 분쟁 해결 | 1차 중재 (트빌리시 중재원) → 2차 조지아 법원 | 저비용 우선 |
| 11. 준거법 | 조지아법 | 사업 소재지 |

**투자자 보호 조항**:
- 대표가 사업을 양도/매각 시 잔여 대출금 즉시 상환
- 분기별 간이 재무 보고서 제출 의무
- 사업 폐업 시 잔여 자산에서 대출 원리금 우선 변제

**대표(차용인) 보호 조항**:
- 대여인은 경영에 일체 간섭할 수 없음
- 상환 유예 조건 명시 (BEP 70% 미달 시)
- 불가항력 시 상환 의무 정지
- 대표 급여($3,000)는 상환보다 우선 지급

### 📋 Option B: 지분투자 계약 — 법적 구조

**법적 형태**: LLC (შეზღუდული პასუხისმგებლობის საზოგადოება) + 주주간 계약 (Shareholders' Agreement)

**LLC 정관 핵심 사항**:

| 정관 조항 | 내용 |
|----------|------|
| 회사명 | K-Street Bites LLC |
| 주주 구성 | 대표 77% / 투자자 23% |
| 자본금 | $20,000 (대표 현물+현금 $15,400 / 투자자 현금 $4,600)* |
| 의결권 | 지분비율 비례 (그러나 주주간 계약으로 경영권 분리) |
| 이사회 | 단독 이사: 대표 (Director 1인) |
| 대표이사 | 대표 단독 |

> *밸류에이션 $67,000 기준, 투자자 $20,000으로 23% 취득. 실제 출자금 배분은 주주간 계약으로 정함

**주주간 계약 (SHA) 핵심 조항**:

| 조항 | 내용 | 보호 대상 |
|------|------|:--------:|
| **1. 경영권** | 대표가 100% 일상 경영권 보유. 이사 선임/해임권 대표 단독 | 대표 |
| **2. 중대사항 합의** | 아래 사항에 한해 투자자 동의 필요 | 투자자 |
| | a) 자본금 20% 이상 단일 지출 | |
| | b) $5,000 이상 신규 부채 발생 | |
| | c) 2호점 이상 확장 | |
| | d) 신규 투자자 유치 (지분 희석) | |
| | e) 사업 형태 변경, 합병, 매각 | |
| **3. 대표 급여** | 월 $3,000 고정 (OPEX로 계상). 배당과 별개 | 대표 |
| **4. 급여 이연** | 현금흐름 악화 시 급여 축소/이연 가능하나, 누적 이연금은 부채로 기록 | 대표 |
| **5. 배당 정책** | 누적 흑자 전환 후, 순이익의 20~30% 반기 배당 | 양측 |
| **6. 안티 딜루션** | 신규 투자 유치 시 투자자에게 우선 참여권(Pro-rata right) | 투자자 |
| **7. 태그얼롱** | 대표 지분 매각 시 투자자도 동일 조건 매각 가능 | 투자자 |
| **8. 드래그얼롱** | 75% 이상 주주 동의 시 전체 지분 일괄 매각 가능 | 대표 |
| **9. Buy-back** | Year 3 이후 대표가 투자자 지분 매입 가능 (투자원금의 2~3배) | 양측 |
| **10. 철수 조건** | 6개월 연속 BEP 70% 미달 시 축소/철수 합의 | 양측 |
| **11. 잔여재산 분배** | 청산 시: ① 부채 변제 → ② 이연 급여 지급 → ③ 투자 원금 반환 → ④ 잔여 지분비례 | 양측 |
| **12. 비경쟁 조항** | 투자자는 동일 시장(트빌리시 한식)에서 3년간 경쟁 사업 금지 | 대표 |
| **13. BAM 조항** | 양 당사자가 BAM 원칙에 동의, 순이익의 3~5%를 사회적 목적에 사용 | BAM |
| **14. 보고 의무** | 월간: 간이 P&L + BAM 리포트 / 반기: 정식 재무제표 + 투자자 미팅 | 양측 |
| **15. 분쟁 해결** | 1차 기도 + 대화 → 2차 기독교 중재 → 3차 트빌리시 중재원 | BAM |

### ⚠️ 법적 주의사항

1. **공증**: 투자 계약서는 조지아 공증인 공증 **강력 권장** (비용 200~500 GEL)
2. **언어**: 계약서는 **영어 + 조지아어** 이중 작성 (법적 분쟁 시 조지아어 우선)
3. **등록**: LLC 정관 변경은 NAPR(공공등기청)에 등기 필수
4. **Work Permit**: 2024년 법 개정으로 대표의 **노동허가 필수** (110 GEL, 연간 갱신)
5. **에스토니아 모델**: LLC는 미배당 이익에 대해 0% 세금. 배당 시에만 15%. 이 점을 투자자에게 명확히 설명

## 1.3 📊 IR-RELATIONS 에이전트 (투자관계)

### 🎯 핵심 발언

> "BAM 투자자는 일반 투자자와 다릅니다. 재무 수익뿐 아니라 '영적 ROI'를 함께 추구합니다. 따라서 Option B(지분투자)가 BAM 투자자에게 압도적으로 매력적입니다."

### 📊 투자자 유형별 매력도

| 투자자 유형 | Option A (대출) 매력도 | Option B (지분투자) 매력도 |
|-----------|:-------------------:|:------------------------:|
| 일반 금융 투자자 | ⭐⭐⭐ | ⭐⭐ |
| BAM 선교 동역자 | ⭐⭐ | ⭐⭐⭐⭐⭐ |
| 교회/선교단체 | ⭐⭐⭐ | ⭐⭐⭐⭐ |
| 가족/지인 | ⭐⭐⭐⭐ | ⭐⭐⭐ |

### 📋 Win-Win 포인트

**Option A — 투자자 Win**: 
- 확정 수익 (연 6% = 3년간 $3,600+ 이자)
- 원금 보장 구조
- 관계 단순 → 리스크 제한적

**Option A — 대표 Win**:
- 100% 경영권
- IE 1% 세율 유지
- 상환 완료 후 완전한 독립

**Option B — 투자자 Win**:
- 높은 잠재 수익 (5년 누적 배당 $18,000+ 지분 가치 상승)
- BAM 팀원으로 참여하는 사명적 보람
- 정기 보고 + 투명한 운영 참관

**Option B — 대표 Win**:
- 월 상환 부담 없음
- 실패 시 상환 의무 없음 (리스크 공유)
- 기도 동역자 + 전략 자문 → 혼자가 아닌 팀
- BAM 사명 강화
