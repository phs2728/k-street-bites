## 🧮 역산 계산: 1년 상환을 위한 필요 매출 (영업일 기준)

### Step 1: 연간 필요 현금
```
연간 운영비 = 8,700 × 12 = 104,400 USD
투자금 상환 = 22,000 USD
───────────────────────────────────
총 필요 현금 = 126,400 USD
```

### Step 2: 필요 매출 계산 (영업일 318일)
```
공헌이익 = 매출 × 65%
필요 공헌이익 = 126,400 USD

필요 연간 매출 = 126,400 / 0.65 = 194,462 USD
필요 월평균 매출 = 194,462 / 12 = 16,205 USD
필요 일평균 매출 = 16,205 / 27일 = 600 USD (영업일 기준)
```

### Step 3: 필요 일일 주문 건수
```
객단가 = 8.3 USD
필요 일일 주문 = 600 / 8.3 = 약 72건 (영업일 기준)

※ 기존 계산 (30일 기준): 65건
※ 수정 계산 (27일 기준): 72건 (+7건 상향 필요)
```

### Step 4: 영업일 감소 보완 전략
```
Option A: 일 목표 72건으로 상향 (+11%)
Option B: 객단가 상향 (23 GEL → 25 GEL, +9%)
Option C: 대표 추가 근무 (공휴일 일부 운영)
Option D: B2B 비중 확대 (평일 안정 수요)

→ 권장: Option B + D 병행
  - 세트 메뉴로 객단가 25 GEL
  - B2B 30% 목표 (휴일 무관 정기 납품)
```
