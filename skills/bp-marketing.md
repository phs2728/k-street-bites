# Skill: /bp-marketing (마케팅 전략 — STP/4P/디지털/팬덤/CRM/B2B)

## Purpose
STP 전략 및 4P 마케팅믹스 기반의 종합 마케팅 전략 수립,
K-Culture 팬덤, CRM, B2B 포함

## Activated Agents
- **Primary**: PERFORMANCE-MARKETING, CRM-SPECIALIST
- **Support**: B2B-SALES, AI-CS, MENU-RD, MACRO-ANALYST

## Execution Steps

### Step 1: STP 전략 수립
```
PERFORMANCE-MARKETING + MACRO-ANALYST 실행:
1. Segmentation (시장 세분화):
   - 인구통계적 (연령, 소득, 국적)
   - 심리적 (K-Culture 관여도, 라이프스타일)
   - 행동적 (배달앱 이용빈도, 외식 빈도)
   - 지리적 (트빌리시 구역별)
2. Targeting (타겟 선정):
   - 1차: K-Culture 팬덤 (18-30세, 높은 구매력)
   - 2차: 외국인 커뮤니티 (주재원, 디지털노마드)
   - 3차: 현지 푸디/트렌드세터
   - 4차: 가족 단위 소비자
3. Positioning (포지셔닝):
   - 포지셔닝 스테이트먼트
   - 차별화 요소 (직접 뽑는 생떡)
   - 지각 맵 (가격 × 프리미엄)
```

### Step 2: 4P 마케팅 믹스
```
PERFORMANCE-MARKETING 실행:
1. Product (제품):
   - 핵심/실질/확장 제품 3단계 정의
   - 메뉴 구성 (Core 70% / Fusion 20% / Seasonal 10%)
2. Price (가격):
   - 가격 전략: 원가가산, 경쟁기반, 가치기반
   - B2C vs B2B 가격 체계
3. Place (유통):
   - 배달앱 (Glovo, Wolt)
   - 오프라인 매장
   - 자체 주문 웹사이트
   - B2B 직납
4. Promotion (촉진):
   - ATL: SNS 광고, 검색광고
   - BTL: 인플루언서, 시식 이벤트
   - PR: 미디어 보도, K-Food 행사
```

### Step 3: 디지털 마케팅 계획
```
PERFORMANCE-MARKETING 실행:
1. 채널별 전략 (Instagram, TikTok, Facebook, Google)
2. 콘텐츠 캘린더 (월별)
3. 광고 예산 배분 및 ROAS 목표
4. 인플루언서 협업 계획
```

### Step 4: K-Culture 팬덤 마케팅
```
PERFORMANCE-MARKETING 실행:
1. K-Drama/K-Pop 연계 이벤트
2. 커뮤니티 구축 전략
3. 체험 마케팅 (한국 요리 클래스)
4. 한국 명절 프로모션
```

### Step 5: CRM 및 리텐션
```
CRM-SPECIALIST 실행:
1. 고객 세그먼테이션 모델 (RFM)
2. 로열티 프로그램 설계
3. 리텐션 자동화 시나리오
4. CLV 극대화 전략
```

### Step 6: B2B 마케팅
```
B2B-SALES 실행:
1. B2B 타겟 리스트
2. 납품 제안서 템플릿
3. B2B 프로모션 전략
4. 샘플링 프로그램
```

### Step 7: 고객 서비스 전략
```
AI-CS 실행:
1. AI 챗봇 대화 설계
2. 다국어 응대 체계
3. 클레임 처리 프로세스
4. 고객 만족도 측정 체계
```
