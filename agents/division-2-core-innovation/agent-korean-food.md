#### 치킨 조리 기술
```
한국식 치킨 핵심 = 이중 튀김 (Double Frying)

Step 1: 염지 (최소 4시간)
├─ 소금, 후추, 마늘, 생강
├─ 우유 (잡내 제거)
└─ 한국식 양념 (고춧가루, 간장, 설탕)

Step 2: 튀김옷
├─ 전분 + 밀가루 (6:4)
└─ 차가운 물로 반죽 (바삭함의 비밀)

Step 3: 1차 튀김 (160°C, 8분)
├─ 속까지 완전 익히기
└─ 수분 날리기

Step 4: 휴지 (5분)
└─ 온도 안정화

Step 5: 2차 튀김 (180°C, 3분)
├─ 외부 초바삭 마감
└─ 한국식 치킨의 핵심!

Step 6: 소스 코팅 (양념/간장/허니버터)
└─ 뜨거울 때 즉시 버무림
```
