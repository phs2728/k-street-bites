## 🇬🇪 조지아 음식 문화 전문 지식

### 1. 조지아 식문화 핵심 이해

#### 수프라 (სუფრა) — 조지아 연회 문화
```
수프라는 조지아 문화의 핵심

특징:
├─ 타마다 (Tamada, 연회 진행자)가 건배 주도
├─ 다양한 요리를 한 상에 차림
├─ 와인과 함께 (조지아는 8,000년 와인 역사)
├─ 가족/친구와 나눔 중심
└─ 3시간+ 오랜 시간 식사

K-Street Bites 적용:
├─ 패밀리/파티 세트 메뉴 = 미니 수프라 컨셉
├─ 공유 가능한 메뉴 구성 (떡볶이, 치킨, 김밥)
├─ 세트에 와인 페어링 가이드 동봉
└─ "K-수프라" 브랜딩으로 문화 교류
```

| **허브 (მწვანილი)** | ⭐⭐⭐⭐⭐ | 허브 도시락 반찬, 허브 치킨 | 현지 허브 적극 활용 |
| **치즈 (ყველი)** | ⭐⭐⭐⭐⭐ | 치즈 떡볶이, 치즈 김밥 | 술구니 치즈 활용 |
| **그릴/바비큐** | ⭐⭐⭐⭐ | 불고기, 양념 치킨 | 숯불 향 강조 |

### 2. 조지아 핵심 재료 & 한식 퓨전 가이드

#### 술구니 치즈 (სულგუნი)
```
조지아 대표 치즈 — 모짜렐라와 비슷하지만 더 짭짤

특성:
├─ 반경질 치즈, 늘어나는 식감
├─ 숙성/훈제 버전 있음
├─ 가격: 약 8-12 GEL/kg (저렴)
└─ 조지아인의 소울 푸드

한식 퓨전 적용:
├─ 치즈 떡볶이 → 술구니 떡볶이 (모짜렐라 대체)
├─ 치즈 김밥 → 술구니 김밥
├─ 치즈 핫도그 → 술구니 핫도그
├─ 치킨 위 토핑 → 술구니 치킨
└─ 하차푸리식 떡 피자
```

#### 호두 (ნიგოზი)
```
조지아 요리의 핵심 재료 — 모든 곳에 호두!

대표 요리:
├─ 바지 (ბაჟე) — 호두 소스 (치킨/생선에 사용)
├─ 파크라리 바드리자니 — 호두 소스 가지말이
├─ 고지나키 (გოზინაქი) — 호두 캔디 (명절 디저트)
└─ 사치비 (საცივი) — 호두 소스 닭요리

한식 퓨전 적용:
├─ 호두 인절미 (디저트)
├─ 호두 소스 비빔밥 (고추장 대안)
├─ 호두 크러스트 치킨 (프리미엄)
├─ 호두 떡갈비
└─ 호두 강정
```

#### 타라곤/코리앤더/바질 (조지아 허브)
```
조지아 요리는 신선한 허브가 핵심

주요 허브:
├─ 타라곤 (ტარხუნა) — 상큼, 살짝 달달
├─ 고수/코리앤더 (ქინძი) — 한국 깻잎 느낌으로 대체 가능
├─ 바질 (რეჰანი) — 향긋
├─ 딜 (კამა) — 상큼
└─ 민트 (პიტნა) — 상쾌

한식 퓨전 적용:
├─ 허브 김치 (타라곤/딜 추가)
├─ 허브 치킨 (조지아식 양념)
├─ 허브 김밥 (코리앤더 + 깻잎)
├─ 허브 반찬 (나물에 현지 허브)
└─ 타라곤 레모네이드 (사이드 음료)
```

#### 조지아 와인 🍷
```
세계에서 가장 오래된 와인 문화 (8,000년)

페어링 가이드:
├─ 사페라비 (Saperavi, 레드) + 양념 치킨, 떡갈비
├─ 르카치텔리 (Rkatsiteli, 화이트) + 김밥, 도시락
├─ 퀴시 (Qvevri, 앰버) + 떡볶이, 분식
└─ 차차 (Chacha, 증류주) + 치킨 안주

K-Street Bites:
├─ 와인 페어링 가이드 카드 동봉 (비용 $0)
├─ "치맥" 대신 "치와" (치킨+와인) 컨셉
└─ 조지아 와인 문화를 존중하면서 한국 치맥 소개
```

### 4. 조지아인 입맛 맞춤 전략

#### 매운맛 관리
```
⚠️ 조지아인은 매운맛에 약합니다!

매운맛 단계 시스템:
├─ 🟢 Mild (არ ცხარე): 고추장 1 — 대부분 조지아인 OK
├─ 🟡 Medium (ცოტა ცხარე): 고추장 2 — 모험적인 조지아인
├─ 🔴 Spicy (ცხარე): 고추장 3 — K-Food 팬, 한국인
└─ 🔥 Extra Spicy (ძალიან ცხარე): 고추장 4 — 챌린지용

기본 설정: 🟢 Mild (조지아인 친화)
→ 떡볶이 기본을 순한맛으로 설정
→ "매운맛 추가" 옵션으로 제공
→ "매운맛 챌린지" 이벤트로 SNS 바이럴
```
