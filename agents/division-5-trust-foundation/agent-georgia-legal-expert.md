### 2. 세금 (Tax Code) 상세

#### 사업자 유형별 세금
| 유형 | 매출 기준 | 세율 | 특징 |
|------|----------|:----:|------|
| **IE (개인사업자)** | < 500,000 GEL/년 | **1%** | 가장 유리 |
| Small Business | < 500,000 GEL/년 | 1% | IE와 유사 |
| Micro Business | < 30,000 GEL/년 | 0% | 무세금 |
| LLC | 제한 없음 | 15% | 배당 시 5% 추가 |

#### IE (Individual Entrepreneur) 상세
```
매출 기준: 연 500,000 GEL 이하 (약 $180,000)
세율: 매출의 1% (순이익 아님!)
신고: 월별 또는 분기별
장점: 단순, 저세율, 외국인 가능
단점: 무한책임, 규모 제한
```

#### 사회보장 기여금 (Pension)
| 항목 | 비율 | 부담자 |
|------|:----:|--------|
| 고용주 연금 기여 | **2%** | 회사 |
| 근로자 연금 기여 | **2%** | 급여 공제 |
| 정부 보조 | 2% | 정부 |

> ⚠️ **한국과 비교**: 4대보험 없음 (약 9.6% → 2%만)

### 3. 식품안전법 (Food Safety) 규정

#### National Food Agency (NFA) 요구사항
| 항목 | 요구 수준 | 비용 (예상) |
|------|----------|------------|
| 사업장 등록 | 필수 | 무료 |
| 식품안전 인증서 | 필수 | 50-200 GEL |
| HACCP 인증 | 선택* | 500-2,000 GEL |
| 위생 교육 | 필수 | 100 GEL/인 |
| 정기 검사 | 연 1-2회 | 무료 |

*소규모 외식업은 간이 HACCP 적용

#### 식품 라벨링 규정
- **조지아어 필수** (ქართულად)
- 성분 표시 (알레르기 포함)
- 유통기한 표시
- 원산지 표시 (수입 식재료)
- 영양정보 (선택)

#### 배달 식품 규정
```
온도 관리:
- 뜨거운 음식: 65°C 이상 유지
- 차가운 음식: 5°C 이하 유지
- 배달 시간: 2시간 이내 권장

포장 요건:
- 식품용 용기 사용
- 밀봉 필수
- 업체명/연락처 표시
```

### 4. 외국인 사업자 규정

#### 🚨 노동허가 (Work Permit) — 2024년 신규 시행

> **⚠️ 중요 법 변경**: 2024년 「노동이주에 관한 유기법(Organic Law on Labor Migration)」 시행으로, 외국인이 조지아에서 취업 또는 사업 활동을 하려면 **노동허가(Work Permit)** 취득이 의무화됨.

| 항목 | 내용 |
|------|------|
| **법적 근거** | Organic Law on Labor Migration (2024) |
| **적용 대상** | 조지아에서 일하는 모든 외국인 (자영업자 포함) |
| **신청 기관** | LEPL — Public Service Development Agency (PSDA) |
| **발급 기간** | 약 10~30 영업일 |
| **유효 기간** | 1년 (갱신 가능) |

##### 비용 상세
| 항목 | 비용 (GEL) | 비용 (USD) | 비고 |
|------|:---------:|:---------:|------|
| 노동허가 신청비 (일반) | **110** | **$40** | 표준 절차 |
| 노동허가 신청비 (급행) | **210** | **$76** | 단축 절차 |
| 서류 번역/공증 | 100~200 | $36~72 | 한국 서류 조지아어 번역 |
| 건강검진 | 50~100 | $18~36 | 현지 병원 |
| 범죄경력증명 (한국 발급) | 무료 | 무료 | 경찰청 온라인 발급 |
| 변호사 대행 (선택) | 200~500 | $72~180 | 직접 신청 시 불필요 |
| **합계 (직접 신청)** | **260~510** | **$94~184** | |
| **합계 (대행 포함)** | **460~1,010** | **$166~365** | |

##### 필요 서류
1. 여권 사본 + 여권 사진
2. 사업자 등록 증명 (IE 또는 LLC)
3. 사업장 임대계약서
4. 범죄경력 증명서 (한국 발급, 아포스티유 필요)
5. 건강검진서 (현지 발급)
6. 신청서 (조지아어)

##### 갱신 절차
- 만료 1개월 전 갱신 신청
- 비용: 초회와 동일 (110 GEL)
- 갱신 시 소득 증빙 필요

> **주의**: 비자(체류)와 노동허가(취업)는 별개입니다.
> - 한국인은 무비자 1년 체류 가능 → 비자 불필요
> - 그러나 **일(사업)을 하려면 노동허가는 반드시 필요**

#### 거주 허가 (Residence Permit)
| 유형 | 기간 | 조건 |
|------|:----:|------|
| 단기 체류 | 1년 | 비자 불필요 (한국) |
| 투자 거주 | 영구 | $300,000+ 투자 |
| 사업 거주 | 1년 | 사업자 등록 + 소득 증빙 |
| 가족 거주 | 1년 | 가족 결합 |

#### 한국인 특례
```
한국-조지아 무비자 협정:
- 입국 시 1년 체류 자동 부여
- 출국 후 재입국 시 리셋

⚠️ 2024년 법 변경 주의:
- 체류(비자): 1년 무비자 → 변동 없음
- 취업/사업: 노동허가(Work Permit) 필수 (신규!)
- 비용: 약 $100~$200 (직접 신청 시)

→ 체류는 자유, 일하려면 노동허가 필요!
```

#### 외국인 법인 설립
| 항목 | 내용 |
|------|------|
| 외국인 100% 소유 | 가능 |
| 최소 자본금 | 없음 |
| 설립 기간 | 1-2일 |
| 비용 | 50-200 GEL |
| 은행 계좌 | 외국인 가능 |

### 6. 실무 행정 절차

#### IE (개인사업자) 등록 절차
```
Step 1: 서류 준비
  ├─ 여권 사본
  ├─ 거주지 증명 (임대계약/호텔)
  └─ 신청서 (조지아어)

Step 2: 공공서비스청 (House of Justice) 방문
  ├─ 위치: 트빌리시, 바투미 등 전국
  ├─ 소요: 30분-1시간
  └─ 비용: 20 GEL (급행 50 GEL)

Step 3: 세무서 (Revenue Service) 등록
  ├─ 자동 연동 또는 별도 신청
  ├─ 소요: 1-2일
  └─ 비용: 무료

Step 4: 은행 계좌 개설
  ├─ Bank of Georgia, TBC Bank 추천
  ├─ 소요: 당일
  └─ 비용: 무료-20 GEL/월
```

#### 식품사업 등록 절차
```
Step 1: IE 등록 (위 참조)

Step 2: National Food Agency 신고
  ├─ 온라인 가능 (nfa.gov.ge)
  ├─ 소요: 3-5일
  └─ 비용: 무료

Step 3: 사업장 검사
  ├─ NFA 담당자 방문
  ├─ 위생 기준 확인
  └─ 시정 요구 시 수정 후 재검사

Step 4: 운영 개시
  ├─ 배달앱 등록 (Glovo/Wolt)
  └─ 영업 시작
```
